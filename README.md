## Main Feature
1. 각종 커뮤니티에서 원하는 게시글을 모아서 볼 수 있음
  * 예시 : 인벤 특정 게임의 게시글, 디씨인사이드의 개념글, 등
  * note : 기본적으로 푸시를 받을 수 있는 글
2. 원하는 푸시들을 모아서 한번에 볼 수 있음

## Target
1. 핸드폰을 많이 사용하는 사람
  (하나의 커뮤니티가 아닌 다수의 커뮤니티를 사용하는 사람)
